Please read the following instructions carefully
------------------------------------------------
1. After running the batch file, while setting the configuration for Stimulation Parameters,
make sure to change the number of columns to atleast 4:
(StimulatonId, StimulationType (voltage/current),DurationDuration1, amplitude1,DurationDuration2, amplitude2..)
Example Values: (auto, 1, auto, auto)

2. Also, while setting the configuration for Stimulation Triggers parameter, 
make sure to change the number of columns to 5.
(StimulatonId, trigger equation,repeats, anode electrodes, kathode electrodes)
Example Values: (auto, auto, auto ,auto, auto)
